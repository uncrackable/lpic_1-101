Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

dd if=/dev/vda of=/root/backup.mbr bs=512 count=1 - бэкап первых 512 байт загрузочной диска (MBR сектор)

1) Добавить таймаут на старт операционной системы в 10 секунд.
https://help.ubuntu.ru/wiki/grub
vim /etc/default/grub 
setting GRUB_TIMEOUT=10
2) Изменить текст выбора операционной системы.
правим файл /etc/grub.d/30_os_prober, поиск по строке menuenrty
3) Попробовать подключить жесткий диск с Windows и указать вашему загрузчкику на возможность загружаться с добавленного диска.
update-grub - для автопоиска нового загрузочного диска; если не работает, то править файл /etc/grub.d/40_custom, UUID диска можно посмортерть в blkid
4) Установить загрузку Windows опцией по умолчанию.
в файле /etc/default/grub меняем значение переменной GRUB_DEFAULT=2 (пункт меню grub, начинается с 0)
